# vue3 admin template

create by vite + vue3 + typescript

# src 下的目录和文件

1. assets（资源文件）

- css
- img

2. components（公用组件）
3. layout（布局框架）

- index.vue（布局主文件）

4. pages（页面，每个文件夹中的 index.vue 都是主文件）

- login（登录页）

5. router（路由配置）

- index.vue（路由主文件）

6. store（状态管理）
7. types（类型定义）

- global.d.ts（全局类型定义）

8. App.vue（设置主题样式、语言等配置）
